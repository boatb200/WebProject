บริการของเรา

ออกแบบสถาปัตย์/ภายใน

วิเคราะห์ความต้องการ วางคอนเซ็ปต์ เขียนแบบก่อสร้าง พร้อมภาพ 3D

  • สำรวจหน้างาน/วัดพื้นที่
  • Moodboard + Concept
  • แบบสถาปัตย์/เฟอร์นิเจอร์
▶ ดูรายละเอียด

ขอบเขตงานออกแบบ

  • แบบแปลน/รูปด้าน/รูปตัด พร้อมรายการวัสดุและสเปก
  • ภาพ 3D / ภาพเรนเดอร์เบื้องต้น 4–8 มุม ต่อพื้นที่หลัก
  • แบบเฟอร์นิเจอร์บิวท์อิน + รายการประกอบ
  • ชุดแบบยื่นก่อสร้าง/ยื่นเทศบาล (กรณีต้องใช้)
  • คู่มือโทนสี/วัสดุ (Material Board) สำหรับหน้างาน

ก่อสร้าง/รีโนเวท

งานโครงสร้าง–สถาปัตย์–งานระบบ จบครบในที่เดียว

  • ถอดปริมาณ + สเปกวัสดุ
  • ทีมวิศวกรควบคุมงาน
  • รับประกันงานหลังส่งมอบ
▶ ดูรายละเอียด

ขั้นตอนการทำงาน

  • ประเมินหน้างาน + สำรวจโครงสร้างเดิม/ระบบเดิม
  • เสนอราคาแบบแยกรายการ (BOQ) โปร่งใส ตรวจสอบได้
  • วางแผนงานรายสัปดาห์และรายวัน พร้อมเส้นทาง Critical Path
  • ควบคุมคุณภาพวัสดุและงานติดตั้งตามมาตรฐานวิชาชีพ
  • ทดสอบระบบก่อนส่งมอบ + รับประกันงานช่างตามประเภทงาน

งานระบบอาคาร

ไฟฟ้า ประปา แอร์ ความปลอดภัย ตามมาตรฐาน

  • เดินระบบไฟ/สื่อสาร มาตรฐาน
  • สุขาภิบาล/ปรับอากาศ
  • ระบบความปลอดภัย/กล้อง
▶ ดูรายละเอียด

รายการระบบ

  • ระบบไฟฟ้า/ตู้เมน/โหลดเซ็นเตอร์/สายดิน ตามมาตรฐาน วสท./มอก.
  • ระบบปรับอากาศ (แยกส่วน/VRF) พร้อมคำนวณ BTU/ระบบระบายอากาศ
  • ระบบสุขาภิบาล/ท่อน้ำดี–ทิ้ง/บ่อดักไขมัน/ถังเก็บน้ำ
  • ระบบ Fire / สัญญาณเตือน / สปริงเกลอร์ (ตามประเภทอาคาร)
  • CCTV / Access Control / ระบบแลนและไวไฟ

ที่ปรึกษาโครงการ

บริหารงบประมาณ–เวลา ตรวจรับงาน คุมคุณภาพ

  • วางแผนโครงการ/แผนงาน
  • ควบคุมต้นทุน–คุณภาพ
  • รายงานความคืบหน้ารายสัปดาห์
▶ ดูรายละเอียด

บริการที่ปรึกษา

  • ตั้งงบ/วางกรอบสัญญา/กำหนดตัวชี้วัด (KPI) โครงการ
  • ตรวจแบบ/ตรวจสเปก/ช่วยคัดเลือกผู้รับเหมา
  • ตรวจรับงานตามงวด + รายงานภาพถ่ายหน้างาน
  • ทบทวน BOQ/ใบเสนอราคา/ใบแจ้งหนี้ เพื่อความคุ้มค่า
  • บริหารความเสี่ยง/การเปลี่ยนแปลงงาน (Variation Order)

แพ็กเกจเริ่มต้น

▶ รายละเอียดแพ็กเกจเบื้องต้น
  • แพ็กเกจออกแบบ: สำรวจหน้างาน + แบบ 2D/3D + สรุปวัสดุ
  • แพ็กเกจก่อสร้าง: BOQ รายการ, ตารางงาน, รับประกันงาน
  • แพ็กเกจที่ปรึกษา: ตรวจรับงวดงาน + รายงานสัปดาห์ + คุมงบ
